## 项目介绍

* 开黑啦 PHP-EasyDev-SDK 是一款针对开黑啦软件所开发的面向PHP开发者所使用的SDK
  - 开发者可以通过简单的工作做出很棒的功能。
  - 项目MIT开源，所有开发者可共同维护。
  - 根据吐槽中心的最新消息，本SDK预计12月开始开发


<br>

## 如何使用

要克隆（clone），您需要[Git]（https://git-scm.com） 安装在您的计算机上。 从您的命令行：

```bash
# 创建项目文件夹
$ mkdir kaiheila
# 进入项目文件夹
$ cd kaiheila
# 克隆资源
$ git clone https://github.com/hugoshao/Kaiheila_PHP_EasyDev.git
```
